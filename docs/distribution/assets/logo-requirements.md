# Logo Requirements for Directory Submissions

## Cline Marketplace (Required)
- **Format:** PNG
- **Size:** 400x400 pixels
- **Background:** Transparent or solid dark (#0A0A0A)
- **Content:** 4DA logo/wordmark
- **File:** Save as `docs/distribution/assets/logo-400x400.png`

## Smithery
- **Format:** PNG or SVG
- **Size:** Square, minimum 200x200
- **File:** Same asset can be used

## DevHunt
- **Format:** PNG
- **Size:** Square, recommended 256x256+
- **File:** Same asset can be used

## General Directory Submissions
Most directories pull the logo from:
1. The npm package README (if it contains an image)
2. The GitHub repo avatar
3. The website favicon/og:image

## Action Required
Create the 400x400 PNG logo and save it here as `logo-400x400.png`.
The same asset can be reused across all platforms.
