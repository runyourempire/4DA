# 4DA v1.0.0 — All signal. No feed.

4DA is a privacy-first desktop app that surfaces developer-relevant content from across the internet — scored against your actual codebase, running entirely on your machine. No feeds to scroll. No algorithms optimizing for engagement. Just the signals that matter to the code you're writing right now.

## What's Inside

- **5-axis relevance scoring** — every piece of content scored against your codebase using the PASIFA algorithm
- **10 built-in content sources** — Hacker News, Reddit, RSS, GitHub, and more — all fetched locally
- **STREETS Playbook** — 7 modules for developer growth, all included free
- **AI briefings** (Free) — AI-generated daily and weekly intelligence summaries
- **Score Autopsy & intelligence panels** (Signal) — full scoring transparency, knowledge gaps, signal chains
- **33 MCP tools** — deep integration with Claude Code, Cursor, and GitHub Copilot
- **Natively multilingual** — 13 languages, content translation, cross-lingual search, native briefings. [Setup guide](MULTILINGUAL.md)

## Quick Start

1. Download and install
2. Launch 4DA — it auto-discovers your projects in seconds

No account required. No configuration needed.

## Privacy

- Runs 100% locally. Zero telemetry. No analytics. No tracking.
- **BYOK** — bring your own API key (Anthropic, OpenAI, or Ollama for fully offline use)
- API keys are stored on your machine and never transmitted anywhere
