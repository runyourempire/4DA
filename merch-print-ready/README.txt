HOW TO UPLOAD TO PRINTFUL
-------------------------

1. Go to printful.com > My Products > Add Product
2. Choose the blank (e.g., Bella+Canvas 3001)
3. Select color: BLACK
4. Click the print area (front or back)
5. Click "Upload file" and select the .svg file
6. Position and scale to the recommended size above
7. If fonts look wrong: open the SVG in Chrome, screenshot at full
   size, and upload the PNG instead

Note: SVGs use system fonts (Courier New, Arial) so they should
render correctly in most tools. If not, any browser can render
them perfectly — just screenshot at high resolution.
